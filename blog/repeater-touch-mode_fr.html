Conduire avec et-repeater en mode Tactile

Un navigateur de répéteurs adapté aux doigts, couplé au GPS continu — vos répéteurs les plus proches, toujours triés, toujours à un tap pendant que vous conduisez.

Sylvain Deguire (VA2OPS) · LiaisonOS 2.3.1

Un nouveau type de shack mobile

Opérer en mobile a toujours impliqué des compromis — un petit écran, des gants, une main sur le volant, et une radio enfouie dans des menus. LiaisonOS 2.3.1 change cette équation avec deux fonctionnalités conçues pour travailler ensemble : le mode Tactile d'et-repeater et le mode GPS continu.

Le résultat : une tablette montée dans votre véhicule qui connaît exactement votre position, maintient votre liste de répéteurs triée par distance en temps réel, et vous permet de taper une carte pour programmer votre radio — sans toucher un seul menu.

Mode Tactile — conçu pour un écran 7"

L'interface standard d'et-repeater est un tableau — idéal sur un moniteur de bureau, mais peu pratique sur un petit écran tactile en mouvement. Le mode Tactile remplace entièrement le tableau par une grille de cartes : quatre répéteurs par rangée, grandes zones de touche, indicatif et fréquence bien visibles, et un bouton SET en bas de chaque carte pour programmer votre radio instantanément.

Le mode Tactile s'active automatiquement lorsque LiaisonOS est en mode Tactile — configurez-le une fois dans QtDashboard, et toutes les applications compatibles s'adaptent. Aucune configuration par application requise.

Le tiroir de filtres

Le bouton hamburger dans la barre supérieure ouvre un tiroir de 75% de largeur depuis la gauche — assez grand pour l'utiliser confortablement au doigt, assez petit pour voir votre contenu derrière. Le tiroir contient de grands boutons bascule pour la bande (multi-sélection : 2m, 70cm, 1.25m, 6m, 10m), le mode (FM, DMR, D-STAR, YSF, P25, NXDN), la distance maximale (5 km jusqu'à TOUS), et un bouton favoris.

Tapez en dehors du tiroir pour le fermer et appliquer vos filtres instantanément. Pas de bouton de confirmation, pas d'étapes supplémentaires.

Conseil : Avant de partir, réglez votre filtre de bande sur 2m + 70cm et votre distance maximale sur 50 km. Le tiroir mémorise vos sélections de filtres entre les sessions.

Mode GPS continu — toujours à jour

et-repeater trie votre liste par distance depuis votre position. Cette position provient de votre configuration utilisateur LiaisonOS — votre locator Maidenhead ou vos coordonnées GPS. La question est : ces données sont-elles fraîches ?

Avec le mode GPS continu, la réponse est : toujours. Lancez QtGpsSync depuis le bouton Sync dans QtDashboard, passez-le en mode GPS, et il alimente votre position dans LiaisonOS en continu tout au long de la session. Pendant que vous conduisez, vos coordonnées se mettent à jour en temps réel. et-repeater re-trie la grille de cartes automatiquement — les répéteurs les plus proches de là où vous êtes maintenant apparaissent toujours en premier.

Cela fonctionne avec votre téléphone Android exécutant l'application LiaisonGPS via Bluetooth, ou un récepteur GPS série comme le Kenwood TH-D74 ou TH-D75. Aucune connexion internet requise — le signal GPS est traité entièrement sur l'appareil.

Le flux de travail mobile complet

  1. Montez votre tablette dans le véhicule et démarrez LiaisonOS
  2. Activez le mode Tactile depuis QtDashboard — toute l'interface s'adapte
  3. Ouvrez QtGpsSync, passez en mode GPS — votre position est maintenant en direct
  4. Ouvrez et-repeater — grille de cartes, triée par distance, mise à jour pendant que vous conduisez
  5. Tapez le tiroir pour filtrer par bande et distance si nécessaire
  6. Tapez SET sur n'importe quelle carte pour programmer votre radio instantanément

Pas de terminal. Pas de menus. Pas de saisie manuelle de fréquence. Conduisez et opérez.

À venir

Le mode Tactile est une fonctionnalité de plateforme — et-repeater n'est que la première application après QtDashboard à le prendre en charge. En v2.3.2, Pat Winlink reçoit le même traitement avec QtPatWinlink — une application Qt native conçue pour le bureau et le tactile dès le premier jour.
