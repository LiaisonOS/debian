Installation Ventoy sur clé USB — Linux étape par étape

VA2OPS · LiaisonOS

Ce guide vous accompagne dans la procédure complète sous Linux : installation de Ventoy sur votre clé USB avec de l'espace réservé, puis utilisation de GNOME Disques pour créer et nommer les partitions ISO et DATA.

Avant de commencer : Téléchargez Ventoy depuis ventoy.net et extrayez l'archive. Branchez votre clé USB — 64 Go ou plus est recommandé.

Partie 1 — Installer Ventoy

1

Identifier votre clé USB avec lsblk

Ouvrez un terminal et exécutez lsblk. Repérez votre clé par sa taille — dans cet exemple, il s'agit de /dev/sdb, une clé de 28,9 Go. Vérifiez bien avant de continuer — Ventoy efface la totalité du disque.

lsblk — clé USB identifiée comme /dev/sdb (28,9 Go)
2

Lancer Ventoy2Disk avec l'espace réservé

Naviguez dans le dossier Ventoy extrait et lancez la commande d'installation avec le paramètre -r pour réserver de l'espace en fin de disque. La valeur est en Mo — 20480 réserve 20 Go pour votre partition DATA. Remplacez /dev/sdb par votre périphérique réel.

sudo bash Ventoy2Disk.sh -i -r 20480 /dev/sdb
Ventoy2Disk.sh — installation avec 20 480 Mo réservés en fin de disque
3

Confirmer la fin de l'installation

Ventoy demande confirmation deux fois avant d'effacer la clé. Une fois terminé, la sortie confirme que Ventoy a été installé avec succès. L'espace réservé est déjà mis de côté — aucun outil de partition supplémentaire n'est nécessaire pour cette étape.

Ventoy installé avec succès — espace réservé confirmé

Partie 2 — Nommer ISO et créer DATA avec GNOME Disques

Ouvrez GNOME Disques (cherchez « Disques » dans votre menu d'applications). Sélectionnez votre clé USB dans le panneau de gauche.

4

Repérer la clé USB et l'espace libre

Vous verrez la grande partition Ventoy (exFAT), une petite partition VTOYEFI et le bloc Espace libre en fin de disque — ce sont les 20 Go que vous avez réservés.

GNOME Disques — partition Ventoy, VTOYEFI et 21 Go d'espace libre visibles
5

Créer la partition DATA — nom et type de formatage

Cliquez sur le bloc Espace libre, puis sur le bouton + pour créer une nouvelle partition. Dans la boîte de dialogue, nommez le volume DATA et sélectionnez Autre comme type — cela vous permettra de choisir exFAT à l'étape suivante.

Formater le volume — nom DATA, sélectionner Autre pour accéder à l'option exFAT
6

Sélectionner exFAT et créer la partition

Dans la boîte de dialogue de format personnalisé, sélectionnez exFAT — Système de fichiers Flash Windows et cliquez sur Créer. exFAT garantit que la partition DATA est lisible et accessible en écriture sous Windows, Mac et Linux sans pilote supplémentaire.

Format personnalisé — exFAT sélectionné, cliquer sur Créer
7

Renommer la partition Ventoy en ISO

Cliquez sur la grande partition Ventoy exFAT. Cliquez sur l'icône modifier (crayon) → Modifier l'étiquette du système de fichiers. Changez l'étiquette de ventoy à ISO et cliquez sur Modifier. Aucun reformatage n'est nécessaire — Ventoy l'a déjà créée en exFAT.

Modifier l'étiquette du système de fichiers — renommer ventoy en ISO
8

Vérifier la disposition finale

GNOME Disques affiche maintenant trois partitions : ISO (exFAT, la partition principale Ventoy), VTOYEFI (petite partition de démarrage) et DATA (exFAT). Votre clé est prête.

Disposition finale — ISO + VTOYEFI + DATA, tout est en place
Note : Les étiquettes ISO et DATA ne sont pas obligatoires pour LiaisonOS — ce sont des conventions qui facilitent la navigation sur la clé. La partition DATA est sélectionnée manuellement lors du premier démarrage dans et-firstboot, quel que soit son nom.

Dernière étape — Copier votre ISO

Ouvrez la partition ISO dans votre gestionnaire de fichiers et copiez-y votre fichier ISO LiaisonOS. C'est tout — Ventoy parcourt la partition automatiquement au démarrage et présente chaque ISO trouvé dans le menu de démarrage.

Astuce : La partition ISO peut contenir plusieurs versions de LiaisonOS en même temps. Ventoy les affichera toutes dans le menu de démarrage — gardez une version précédente à côté de la dernière comme solution de secours éprouvée.

Votre clé est prête. Démarrez depuis celle-ci et LiaisonOS se lancera directement — aucune installation requise.
