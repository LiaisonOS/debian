Your Ventoy USB Key — The Right Way

How to set up a Ventoy USB key for LiaisonOS — the recommended 3-partition layout, sizing guide, and step-by-step install on Windows, Mac, and Linux.

Sylvain Deguire (VA2OPS) · LiaisonOS 2.2.6

Why Ventoy?

Most USB boot tools overwrite your entire drive with a single ISO. If you want to try a newer version, you flash again. If you want a fallback, you need a second USB key. Ventoy changes that completely.

With Ventoy, you install the bootloader once — then you simply copy ISO files onto the drive. At boot time, Ventoy presents a menu and you pick which ISO to boot. Updating LiaisonOS means dragging the new ISO onto the drive. Keeping the previous version as a fallback means not deleting the old one. No reflashing, ever.

This is why Ventoy is the recommended way to run LiaisonOS.

Choosing your USB key

The USB key is a personal choice that depends on how you plan to use LiaisonOS. Here is a practical guide:

  • 32 GB minimum — workable for basic use, but leaves little room to grow
  • 64 GB — recommended for most users. Enough for a few ISO versions, offline maps for your region, and a solid persistence backup
  • 128 GB or more — strongly recommended for daily users. Room for multiple ISO versions, large map tilesets, Kiwix/Wikipedia ZIM files, Thunderbird email archives, PDF radio manuals, and personal files

Think about what you want to keep: each LiaisonOS ISO is roughly 4.7 GB. A set of offline maps for a region can be several gigabytes. A full Wikipedia ZIM file is large. Thunderbird email over time adds up. When in doubt, go bigger — a quality 128 GB USB key is inexpensive and you will never regret the space.

The recommended 3-partition layout

Rather than leaving the entire drive as one Ventoy partition, we recommend splitting it into three dedicated partitions. This keeps your ISOs, your persistence data, and the Ventoy boot system cleanly separated.

Partition Label Filesystem Purpose Suggested size
1 ISO exFAT Main Ventoy partition — renamed from ventoy. Drop your ISO files here, Ventoy boots from them automatically Bulk of the drive
2 VTOYEFI FAT Ventoy EFI boot partition — created automatically, do not touch ~32 MB (auto)
3 DATA exFAT Persistence backup — where et-persistence saves all your settings, configs, and data 20 GB or more
Why exFAT for DATA? exFAT is readable and writable on Windows, Mac, and Linux without any extra drivers. This means you can access your backed-up files from any computer — important in a field or emergency context.

The ISO partition is where you store your ISO files — the last 3 to 5 versions of LiaisonOS, so you always have a known-good fallback. The DATA partition is where et-persistence saves your data between sessions — WiFi credentials, Bluetooth pairings, radio configs, logs, email, maps, and personal files. More on this in the next article.

Step-by-step: Windows

  1. Download Ventoy for Windows and extract the zip file.
  2. Insert your USB key and run Ventoy2Disk.exe as Administrator.
  3. Select your USB drive from the dropdown — double-check the drive letter, Ventoy will erase it.
  4. Click Install and confirm. Ventoy creates the boot partition and a large exFAT data partition.
  5. Open Disk Management (right-click Start → Disk Management).
  6. Find the Ventoy exFAT partition. Right-click → Shrink Volume. Shrink it to your target ISO size (e.g. 32768 MB for 32 GB).
  7. Right-click the new unallocated space → New Simple Volume. Format as exFAT, label it ISO.
  8. The remaining space becomes DATA — create another new simple volume, exFAT, labeled DATA.
  9. Copy your LiaisonOS ISO file into the ISO partition.
Warning: Always verify the drive letter before clicking Install in Ventoy. Selecting the wrong drive will erase it completely with no undo.

Step-by-step: Linux

  1. Download Ventoy for Linux and extract the archive.
  2. Identify your USB drive with lsblk — look for the correct size.
  3. Run: sudo bash Ventoy2Disk.sh -i -r 20480 /dev/sdb (replace sdb with your actual drive). The -r flag reserves space at the end in MB — 20480 = 20 GB.
  4. Open GNOME Disks, create the DATA partition (exFAT) in the free space, then rename the Ventoy partition label to ISO.
  5. Copy your LiaisonOS ISO into the ISO partition.

Warning: On Linux, lsblk before running Ventoy to confirm the correct device. Running Ventoy on the wrong device (/dev/sda is usually your internal hard drive) will erase it.

Managing your ISO files

Once your key is set up, managing LiaisonOS versions is simple:

  • Adding a new version — download the new ISO and copy it to the ISO partition. Ventoy picks it up automatically at next boot.
  • Keeping older versions — just leave the old ISO on the drive. At boot, Ventoy shows all available ISOs and you choose which one to start. This is your safety net.
  • Removing old versions — delete the ISO file when you no longer need it. Simple as that.
Recommendation: Keep the last 3 to 5 ISO versions on your ISO partition. With LiaisonOS at ~4.7 GB per version, that is 15–24 GB — comfortable on a 32 GB ISO partition and leaving room to spare.
