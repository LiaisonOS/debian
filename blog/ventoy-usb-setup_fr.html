Votre clé USB Ventoy — La bonne façon

Comment configurer une clé USB Ventoy pour LiaisonOS — la disposition recommandée en 3 partitions, guide de dimensionnement et installation pas à pas sur Windows, Mac et Linux.

Sylvain Deguire (VA2OPS) · LiaisonOS 2.2.6

Pourquoi Ventoy ?

La plupart des outils de création de clé USB bootable écrasent l'intégralité de votre clé avec une seule image ISO. Pour essayer une version plus récente, vous devez re-flasher. Pour garder une version de secours, il vous faut une deuxième clé. Ventoy change tout cela.

Avec Ventoy, vous installez le chargeur d'amorçage une seule fois — puis vous copiez simplement des fichiers ISO sur la clé. Au démarrage, Ventoy affiche un menu et vous choisissez quel ISO démarrer. Mettre à jour LiaisonOS revient à glisser le nouvel ISO sur la clé. Garder la version précédente en secours revient à ne pas supprimer l'ancien fichier. Plus jamais de re-flash.

C'est pourquoi Ventoy est la méthode recommandée pour utiliser LiaisonOS.

Choisir votre clé USB

La clé USB est un choix personnel qui dépend de la façon dont vous comptez utiliser LiaisonOS. Voici un guide pratique :

  • 32 Go minimum — utilisable pour un usage de base, mais peu de marge pour grandir
  • 64 Go — recommandé pour la plupart des utilisateurs. Suffisant pour quelques versions ISO, des cartes hors ligne pour votre région et une sauvegarde de persistance solide
  • 128 Go ou plus — fortement recommandé pour les utilisateurs quotidiens. De la place pour plusieurs versions ISO, de grands jeux de tuiles cartographiques, des fichiers ZIM Kiwix/Wikipédia, des archives de courriels Thunderbird, des manuels radio PDF et des fichiers personnels

Réfléchissez à ce que vous voulez conserver : chaque ISO LiaisonOS pèse environ 4,7 Go. Un ensemble de cartes hors ligne pour une région peut représenter plusieurs gigaoctets. Un fichier ZIM Wikipédia complet est volumineux. Les courriels Thunderbird s'accumulent au fil du temps. Dans le doute, prenez plus grand — une clé USB de qualité 128 Go est peu coûteuse et vous ne regretterez jamais l'espace disponible.

La disposition recommandée en 3 partitions

Plutôt que de laisser la clé entière comme une seule partition Ventoy, nous recommandons de la diviser en trois partitions dédiées. Cela garde vos ISO, vos données de persistance et le système de démarrage Ventoy proprement séparés.

Partition Nom Système de fichiers Rôle Taille suggérée
1 ISO exFAT Partition principale Ventoy — renommée depuis ventoy. Déposez vos ISO ici, Ventoy les détecte automatiquement au démarrage Majeure partie de la clé
2 VTOYEFI FAT Partition de démarrage EFI Ventoy — créée automatiquement, ne pas toucher ~32 Mo (auto)
3 DATA exFAT Sauvegarde de persistance — où et-persistence sauvegarde tous vos paramètres et données Reste de la clé
Pourquoi exFAT pour DATA ? L'exFAT est lisible et inscriptible sur Windows, Mac et Linux sans pilotes supplémentaires. Cela signifie que vous pouvez accéder à vos fichiers sauvegardés depuis n'importe quel ordinateur — important dans un contexte terrain ou d'urgence.

La partition ISO est là où vous stockez vos fichiers ISO — les 3 à 5 dernières versions de LiaisonOS, pour toujours avoir une version de secours fonctionnelle. La partition DATA est là où et-persistence sauvegarde vos données entre les sessions — identifiants WiFi, jumelages Bluetooth, configurations radio, journaux, courriels, cartes et fichiers personnels. Plus de détails dans le prochain article.

Pas à pas : Windows

  1. Téléchargez Ventoy pour Windows et extrayez le fichier zip.
  2. Insérez votre clé USB et lancez Ventoy2Disk.exe en tant qu'Administrateur.
  3. Sélectionnez votre clé USB dans la liste déroulante — vérifiez deux fois la lettre de lecteur, Ventoy va l'effacer.
  4. Cliquez sur Install et confirmez. Ventoy crée la partition de démarrage et une grande partition exFAT.
  5. Ouvrez la Gestion des disques (clic droit sur Démarrer → Gestion des disques).
  6. Trouvez la partition exFAT de Ventoy. Clic droit → Réduire le volume. Réduisez-la à la taille cible pour ISO (ex. : 32 768 Mo pour 32 Go).
  7. Clic droit sur l'espace non alloué → Nouveau volume simple. Formatez en exFAT, nommez-le ISO.
  8. L'espace restant devient DATA — créez un nouveau volume simple, exFAT, nommé DATA.
  9. Copiez votre fichier ISO LiaisonOS dans la partition ISO.
Attention : Vérifiez toujours la lettre de lecteur avant de cliquer sur Install dans Ventoy. Sélectionner le mauvais lecteur l'effacera complètement sans possibilité d'annulation.

Pas à pas : Linux

  1. Téléchargez Ventoy pour Linux et extrayez l'archive.
  2. Identifiez votre clé USB avec lsblk — repérez la bonne taille.
  3. Exécutez : sudo bash Ventoy2Disk.sh -i -r 20480 /dev/sdb (remplacez sdb par votre périphérique réel). Le paramètre -r réserve de l'espace en fin de disque en Mo — 20480 = 20 Go.
  4. Ouvrez GNOME Disques, créez la partition DATA (exFAT) dans l'espace libre, puis renommez l'étiquette de la partition Ventoy en ISO.
  5. Copiez votre ISO LiaisonOS dans la partition ISO.

→ Guide Linux pas à pas avec captures d'écran

Attention : Sur Linux, utilisez lsblk avant de lancer Ventoy pour confirmer le bon périphérique. Exécuter Ventoy sur le mauvais périphérique (/dev/sda est généralement votre disque dur interne) l'effacera.

Gérer vos fichiers ISO

Une fois la clé configurée, gérer les versions de LiaisonOS est simple :

  • Ajouter une nouvelle version — téléchargez le nouvel ISO et copiez-le dans la partition ISO. Ventoy le détecte automatiquement au prochain démarrage.
  • Conserver les anciennes versions — laissez simplement l'ancien ISO sur la clé. Au démarrage, Ventoy affiche tous les ISO disponibles et vous choisissez lequel démarrer. C'est votre filet de sécurité.
  • Supprimer les vieilles versions — supprimez le fichier ISO quand vous n'en avez plus besoin. Aussi simple que ça.
Recommandation : Gardez les 3 à 5 dernières versions ISO sur votre partition ISO. Avec LiaisonOS à ~4,7 Go par version, cela représente 15–24 Go — confortable sur une partition ISO de 32 Go avec de la marge.
