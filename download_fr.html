Avis de développement continu — LiaisonOS utilise un modèle de publication agile. Chaque version est pleinement fonctionnelle. Démarrez toujours depuis USB d'abord avant une installation permanente. Les modifications récentes peuvent occasionnellement introduire des problèmes inattendus — tester depuis USB avant d'installer est toujours l'approche recommandée.

Nouveautés de la v2.3.1 / v2.3.2

  • v2.3.2 — QtTerminal — LiaisonOS aura son propre terminal poru accueillir Paracon, Chattervox et linBPQ.
  • QtDashboard — Tableau de bord natif C++ — Une étape majeure : l'ancien tableau de bord Python/GTK est remplacé par une application Qt5 native entièrement en C++. Plus rapide, plus propre, conçu spécifiquement pour LiaisonOS. Horloge UTC intégrée avec secondes, sélection de modem directement dans le tableau de bord, configuration radio/CAT, édition du profil utilisateur, et 6 modes récents toujours prêts en un seul tap.
  • Mode Bureau & Mode Tactile — QtDashboard fonctionne en deux modes d'interface distincts. Le mode Bureau est optimisé pour souris et clavier. Le mode Tactile est conçu pour les tablettes 7" et les écrans montés en véhicule — grandes cartes, contrôles adaptés aux doigts, bascule instantanée sans redémarrage. À notre connaissance, cette conception double-mode est unique dans l'univers des OS pour radioamateurs.
  • et-repeater en mode Tactile — Le navigateur de répéteurs s'adapte au mode Tactile avec une grille de cartes, un tiroir de filtres plein écran (bande, mode, distance, favoris) et une barre de recherche en temps réel. Le mode Tactile est une fonctionnalité de plateforme lue depuis user.json — d'autres applications suivront dans les prochaines versions.
  • À venir en v2.3.3 — QtPatWinlink — Pat Winlink réinventé en application Qt native avec support complet du mode Tactile.

Comment installer

1
Téléchargez le fichier ISO avec le bouton ci-dessus.
2
Vérifiez la somme SHA256 pour confirmer que le fichier n'a pas été corrompu lors du téléchargement.
3
Écrivez sur une clé USB avec balenaEtcher (recommandé pour débutants) ou Ventoy (multi-démarrage).
4
Démarrez depuis USB — appuyez sur F12, F2 ou Échap au démarrage pour accéder au menu de démarrage. Sur Mac, maintenez Option.
5
Sélectionnez votre langue (FR/EN) au premier démarrage, configurez votre indicatif et votre radio, et vous êtes sur les ondes.
6
Optionnel : Installez de façon permanente sur votre disque dur via Calamares (icône de l'installateur sur le bureau).

Outils de création de clé USB démarrable

Windows

  • Rufus — rapide, fiable, open-source
  • balenaEtcher — simple en 3 étapes
  • Ventoy — multi-démarrage

macOS

  • balenaEtcher — meilleur choix Mac
  • UNetbootin — multiplateforme
  • dd (Terminal) — intégré

Linux

  • balenaEtcher — AppImage
  • Ventoy — copiez les ISO sur la clé
  • GNOME Disques — pré-installé
  • dd — classique, rapide
